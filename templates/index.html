Materials Database API Guide

Introduction

Welcome to the Materials Database API, designed to manage and analyze material data efficiently.

Setting Up

To set up the API, follow these simple steps:

1. Clone the repository:
    git clone something like this:-> https://github.com/Material-DB/materials-api.git
2. Install dependencies:
    pip install -r requirements.txt
    make sure these versions are correct
    Python 3.10.10
    Flask 2.0.1
    Werkzeug 2.0.0

Running the API

To start the API, run the following command:

python app.py

Visit http://192.168.1.5:8080 in your browser to interact with the API.

  • Home
  • Store Data
  • Search
  • Numerical Filter
  • Boolean Field Filter
  • Element-Based Filtering
  • Multi-Column Sort

Examples:

  • To Store new data, you can use this command:
    • curl -X POST -H "Content-Type: application/json" -d '{"nsites": 1, "formula_pretty": "H2O", "volume": 18.01528}' http://192.168.1.5:8080/store-data
  • To Search:
    • curl -X GET "http://192.168.1.5:8080/search?query=H2O"
  • Element-Based Filtering:
    • curl -X GET "http://192.168.1.5:8080/element-filter?elements=Ti-N"
  • Numerical Filter:
    • curl -X GET "http://192.168.1.5:8080/filter?field=volume&operator=>&value=10"
  • Multi-Column Sort:
    • curl -X GET "http://192.168.1.5:8080/sort?columns=volume,density&order=asc"
  • Boolean Field Filter:
    • curl -X GET "http://192.168.1.5:8080/boolean-filter?field=is_stable&value=true"

Objective Details

Store and Export Data

The API allows storing data using the /store-data endpoint. Data is stored in the MongoDB database.

String/Sub-String Search on Text Fields

Perform string/sub-string searches on text fields using the /search endpoint. Specify the query parameter to search for specific terms.

Mathematical Filters on Numerical Fields

Apply mathematical filters (<, >, =) on numerical fields using the /filter endpoint. Specify the field, operator, and value parameters in the URL.

Boolean Field Filter

Filter data based on boolean fields using the /boolean-filter endpoint. Specify the field and value parameters in the URL.

Validate Input for the Database

The API validates input data to ensure it meets the required format. Errors are returned if validation fails.

Multi-Column Sort

Sort data based on multiple columns using the /sort endpoint. Specify the columns and order parameters in the URL.

Element-Based Filtering

Perform element-based filtering using the /element-filter endpoint. Specify elements in the URL (e.g., Ca-O, Ti-N).

Feel free to explore and interact with the API to meet your material data analysis needs!
